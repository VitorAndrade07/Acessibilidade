# Mobile Accessibility & UX Research (Kotlin)
This project explores the implementation of accessibility standards in mobile development using **Kotlin**. It focuses on creating inclusive digital experiences for all users.

### 🛠️ Key Features:
* **Inclusive Design**: Implementation of UI components that follow global accessibility guidelines (WCAG).
* **Native Development**: Built with Kotlin to ensure high performance and seamless integration with Android accessibility services.
* **Logic-Driven UI**: Applied structured logic to handle screen readers, high contrast modes, and scalable touch targets.

### 🧠 Why this matters:
For a Data Analyst or Backend Developer, understanding accessibility is a differentiator. It shows that I build systems with the **end-user in mind**, ensuring that data-driven interfaces are usable by everyone, regardless of their physical abilities.
